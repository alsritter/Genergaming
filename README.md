Game

* move left <kbd>A</kbd>
* move right <kbd>D</kbd>
* Jump <kbd>J</kbd>
* Run <kbd>K</kbd>


Genergaming is an editor that can edit game map generation games online. Once the map is drawn, it can be run online through a WebGL client written in Unity.
